# Sistema de e_commerce API Rest utilizando Laravel

<h1>Documentação</h1>
<h2>Url Base - localhost:8000</h2>
<h3>Clientes - GET </h3>
Endpoint: /cliente
<h5>Retorno</h5> 
  
Retorna todos os clientes

<h3>Clientes - POST </h3>
Endpoint: /cliente
<h5>Body</h5>
nome: String,
email: String,
cpf: String (Somente Numeros),
data_nascimento: String (aaaa-mm-dd)

<h5>Retorno</h5> 
Sucesso: Retorna o cliente criado
